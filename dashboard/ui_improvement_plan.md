# UI Improvement Plan

1. **Phase 1: Foundation & Variables**
    - Update `:root` variables with more vibrant, curated colors.
    - Add new variables for glows and glassmorphism.

2. **Phase 2: Layout & Glassmorphism**
    - Update `.topbar`, `.left-panel`, and `.right-panel` with glass-like styles.
    - Implement mesh gradient background for the whole application.
    - Improve panel borders and spacing.

3. **Phase 3: Component Styling**
    - **Track Cards**: Add premium hover states, inner glows, and smoother transitions.
    - **Buttons**: Upgrade `.btn-eval` and `.btn-secondary` with stronger gradients and better interactions.
    - **Status Items**: Refine the status grid with subtle backgrounds and better typography.
    - **Observation Feed**: Make feed items look more like "log" entries with high contrast.

4. **Phase 4: Visual Polish**
    - Add a global "Scanline" effect for a tactical HUD feel.
    - Refine Leaflet markers and popups for a more integrated look.
    - Implement smoother scrollbars.

5. **Phase 5: Responsive & Final Checks**
    - Ensure everything fits well on different screen sizes.
    - Verify all interactions feel "premium".
